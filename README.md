# AURA • Elite Travel Concierge System

An immersive, production-ready AI agent prototype designed to orchestrate highly tailored, off-market travel curations for elite clientele. Powered by the Google Gemini 2.5 Flash model and an advanced custom Streamlit layout engine.

## 🚀 Architectural Features
- **Deterministic Multi-Turn Logic:** Implements strict guardrails to suppress competitor platform mentions (Expedia, Booking.com, Airbnb) and seamlessly handles multi-stage state evaluations for price objections.
- **Session Memory Architecture:** Utilizes native state preservation mechanisms (`st.session_state`) to retain deep conversational context across processing cycles without losing track of user intent.
- **Glassmorphic UI Engine:** Crafted with a responsive dark aesthetic, featuring modern card designs, hover transforms, custom typographic weights, and a clean interface.

## 🛠️ Tech Stack
- **Core LLM:** Google Gemini 2.5 Flash (`google-genai`)
- **Interface Framework:** Streamlit
- **Configuration Management:** Python-Dotenv
